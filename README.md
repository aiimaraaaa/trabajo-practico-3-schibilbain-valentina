# Los Simpson 

Aplicación web que muestra los personajes de Los Simpson usando una API pública.

## Funcionalidades

- Listado de personajes con imagen, nombre, ocupación y estado
- Buscador en tiempo real por nombre
- Modal con el detalle completo de cada personaje

## Tecnologías utilizadas

- HTML5
- CSS
- JavaScript (async/await, fetch, DOM, arrays, objetos)
- Bootstrap 5.3.3

## API utilizada

- https://thesimpsonsapi.com/api/characters
